Este projeto consiste na criação de uma API focada no tema "Músicas". A API oferece suporte a três gêneros musicais distintos, com seis músicas selecionadas para cada gênero. Ao clicar no botão correspondente a um gênero, a API filtra e retorna apenas as músicas pertencentes ao gênero selecionado. Para cada música, são fornecidos o nome, o cantor e um link para ouvi-la no YouTube.
